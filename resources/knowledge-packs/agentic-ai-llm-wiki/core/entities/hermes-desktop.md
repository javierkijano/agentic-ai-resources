# Hermes Agent Desktop

Hermes Agent Desktop (`Felix-Forever/hermes-agent-desktop`) is a graphical desktop client designed to wrap the core [[hermes]] CLI system.

Unlike the CLI or [[hermes-webui]], its primary goal is enabling "Multi-Agent AI Teams" — orchestrating parallel instances, visualizing task execution, and providing a UI for comprehensive autonomous workflows.
