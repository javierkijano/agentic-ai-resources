# Hermes WebUI

Hermes WebUI (`nesquena/hermes-webui` and forks like `Web3CZ/Web3Hermes`) provides a remote web browser and mobile web interface for interacting with the [[hermes]] CLI agent.

This project enables remote control and interaction with the backend structure of Hermes, facilitating cross-platform access without the need for the local terminal environment.
